# セキュリティチェックリスト

## 認証・認可

- [ ] パスワードはハッシュ化されているか（bcrypt, argon2）
- [ ] セッショントークンは十分なエントロピーがあるか
- [ ] 認証トークンの有効期限は適切か
- [ ] 認可チェックが全てのエンドポイントで行われているか
- [ ] 権限昇格の脆弱性がないか

## 入力バリデーション

- [ ] ユーザー入力は全てサニタイズされているか
- [ ] SQLインジェクション対策（パラメータ化クエリ）
- [ ] XSS対策（出力エスケープ）
- [ ] CSRF対策（トークン検証）
- [ ] パストラバーサル対策

## 通信

- [ ] HTTPS が強制されているか
- [ ] CORS 設定が適切か
- [ ] セキュリティヘッダーが設定されているか
  - X-Content-Type-Options
  - X-Frame-Options
  - Content-Security-Policy

## 機密情報

- [ ] シークレットがコードにハードコードされていないか
- [ ] 環境変数で機密情報を管理しているか
- [ ] ログに機密情報が出力されていないか
- [ ] エラーメッセージに内部情報が含まれていないか

## 依存関係

- [ ] 依存パッケージに既知の脆弱性がないか
- [ ] 使用していない依存関係を削除したか
