# 新メンバー招待テンプレート（コピペ用）

> 新メンバーを招待する際にこのテンプレートを使用してください。
> `{{...}}` の部分をプロジェクト固有の情報に置き換えてください。

---

## Slack/メールで送る文面

```
【プロジェクト参加のご案内】

〇〇さん

プロジェクト「{{PROJECT_NAME}}」への参加ありがとうございます。

■ 最初にやること

1. GitHubリポジトリをクローン
   → {{REPO_URL}}

2. 以下のドキュメントを順番に読む
   - docs/onboarding/ONBOARDING_CHECKLIST.md（初回セットアップ）
   - QUICK_GUIDE.md（全体の流れ）
   - CLAUDE.md（開発ルール）

3. docs/PROJECT_CONFIG.md の「メンバー別設定」に自分の情報を追加

4. ローカル環境で動作確認


■ 重要

メンバー登録が完了するまで、commit・PR はブロックされます。
必ず ONBOARDING_CHECKLIST.md に沿って設定を完了してください。


■ 困ったら

Slackの #{{CHANNEL_NAME}} で質問してください。


よろしくお願いします！
```

---

## チェックリスト（管理者用）

招待前に以下を確認：

- [ ] GitHubリポジトリに招待済み
- [ ] Slack/Discord に招待済み（該当する場合）
- [ ] Vercel チームに招待済み（該当する場合）
- [ ] 上記文面を送付済み

---

## 招待後のフォローアップ

- [ ] 3日以内に `docs/PROJECT_CONFIG.md` にメンバー追加されたか確認
- [ ] 追加されていなければリマインド
