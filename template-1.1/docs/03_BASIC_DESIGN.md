### API詳細（主要なもの）

#### A-001: ログイン

**エンドポイント**: `POST /api/auth/login`

**リクエスト**:
```json
{
  "email": "user@example.com",
  "password": "password123"
}
```

**レスポンス（成功）**:
```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "user": {
    "id": "uuid",
    "email": "user@example.com",
    "name": "ユーザー名"
  }
}
```

**レスポンス（失敗）**:
```json
{
  "error": "Invalid credentials"
}
```
