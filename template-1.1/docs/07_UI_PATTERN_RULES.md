# UIパターンルール集

**目的**: ガーディアンAIがユーザビリティの抜け漏れをチェックするための基準

---

## 基本画面パターン（10種）

### 1. 一覧画面（テーブル/カード）

**必須要素**
- [ ] 各項目に「編集」ボタン
- [ ] 各項目に「削除」ボタン
- [ ] データがゼロ件の時の表示（空状態メッセージ）
- [ ] ローディング表示
- [ ] ページネーション or 無限スクロール（10件以上想定時）

**あると良い**
- [ ] 検索/フィルター機能
- [ ] ソート機能
- [ ] 一括選択/一括操作

---

### 2. 詳細画面

**必須要素**
- [ ] 「戻る」ボタン（一覧への導線）
- [ ] 「編集」ボタン
- [ ] 「削除」ボタン
- [ ] ローディング表示
- [ ] データが存在しない場合のエラー表示

**あると良い**
- [ ] 関連データへのリンク

---

### 3. 作成画面（フォーム）

**必須要素**
- [ ] 「保存」ボタン
- [ ] 「キャンセル」ボタン
- [ ] 必須項目の明示（*マークなど）
- [ ] バリデーションエラー表示（項目ごと）
- [ ] 保存中のローディング表示
- [ ] 保存成功時のフィードバック（トースト等）
- [ ] 保存失敗時のエラー表示
- [ ] 保存成功後の遷移先

**あると良い**
- [ ] 入力途中で離脱時の確認ダイアログ
- [ ] 自動保存（ドラフト）

---

### 4. 編集画面

**必須要素**
- [ ] 作成画面と同じ要素すべて
- [ ] 既存データの初期表示
- [ ] 「変更を破棄」の確認

**あると良い**
- [ ] 変更箇所のハイライト
- [ ] 変更履歴

---

### 5. 削除処理

**必須要素**
- [ ] 削除前の確認ダイアログ
- [ ] 「本当に削除しますか？」のメッセージ
- [ ] 削除対象の名前を表示（何を消すか明確に）
- [ ] 削除中のローディング表示
- [ ] 削除成功時のフィードバック
- [ ] 削除失敗時のエラー表示

**あると良い**
- [ ] 削除の影響範囲の表示（「関連する〇〇も削除されます」）
- [ ] ソフトデリート（復元可能）

---

### 6. ログイン/認証

**必須要素**
- [ ] メールアドレス/パスワード入力欄
- [ ] 「ログイン」ボタン
- [ ] ログイン中のローディング表示
- [ ] ログイン失敗時のエラー表示
- [ ] パスワード表示/非表示トグル
- [ ] 「パスワードを忘れた」リンク

**あると良い**
- [ ] 「ログイン状態を保持」チェックボックス
- [ ] ソーシャルログイン

---

### 7. ダッシュボード

**必須要素**
- [ ] 主要な統計/KPIの表示
- [ ] 各セクションへの導線
- [ ] ローディング表示
- [ ] データがない場合の初期状態表示

**あると良い**
- [ ] 期間フィルター
- [ ] グラフ/チャート
- [ ] 最近のアクティビティ

---

### 8. 設定画面

**必須要素**
- [ ] 「保存」ボタン
- [ ] 保存成功時のフィードバック
- [ ] 変更前の確認（重要な設定の場合）

**あると良い**
- [ ] セクション分け
- [ ] 設定のリセット機能

---

### 9. 検索/フィルター

**必須要素**
- [ ] 検索入力欄
- [ ] 検索実行ボタン or 自動検索
- [ ] 検索中のローディング表示
- [ ] 検索結果ゼロ件の表示
- [ ] フィルターのクリア機能

**あると良い**
- [ ] 検索履歴
- [ ] サジェスト
- [ ] 検索条件の保存

---

### 10. モーダル/ダイアログ

**必須要素**
- [ ] 「閉じる」ボタン（×マーク）
- [ ] 背景クリックで閉じる or 閉じない（意図的に選択）
- [ ] ESCキーで閉じる
- [ ] モーダル内のアクションボタン
- [ ] モーダルのタイトル

**あると良い**
- [ ] フォーカストラップ（タブ移動がモーダル内に留まる）

---

## AI関連パターン（4種）★重点

### 11. チャット画面（AI対話）

**必須要素**
- [ ] メッセージ入力欄
- [ ] 「送信」ボタン
- [ ] 送信中の入力欄の無効化
- [ ] AI応答中のローディング表示（「考え中...」等）
- [ ] ユーザーとAIのメッセージの視覚的区別
- [ ] メッセージのタイムスタンプ
- [ ] 自動スクロール（最新メッセージへ）
- [ ] 会話履歴の表示
- [ ] 新規会話の開始ボタン

**あると良い**
- [ ] メッセージのコピー機能
- [ ] メッセージの再生成機能
- [ ] メッセージの編集機能
- [ ] ストリーミング表示（文字が流れるように表示）
- [ ] 会話のエクスポート
- [ ] 会話の削除
- [ ] ファイル添付機能

---

### 12. AI処理中の表示（ローディング/ストリーミング）★最重要

**必須要素**
- [ ] 処理中であることが明確にわかる表示
- [ ] 推定待ち時間の表示（可能な場合）
- [ ] キャンセルボタン（長時間処理の場合）
- [ ] 処理完了時のフィードバック

**ストリーミング表示の場合**
- [ ] 文字が流れるように表示される
- [ ] 途中でも読み始められる
- [ ] 完了がわかる（カーソル消える等）

**バックグラウンド処理の場合**
- [ ] 「処理中」のステータス表示
- [ ] 進捗バー（可能な場合）
- [ ] 完了時の通知
- [ ] 処理結果への導線

**エラー時**
- [ ] エラーメッセージの表示
- [ ] 再試行ボタン
- [ ] エラーの原因説明（可能な場合）

---

### 13. AI設定画面

**必須要素**
- [ ] 現在の設定値の表示
- [ ] 設定変更のUI（スライダー、セレクトボックス等）
- [ ] 設定の説明文（各項目が何に影響するか）
- [ ] 「保存」ボタン
- [ ] デフォルト値に戻す機能

**よくある設定項目**
- [ ] AIモデル選択（使い分けがある場合）
- [ ] 応答の長さ/詳細度
- [ ] 応答のトーン/スタイル
- [ ] API使用量の上限設定
- [ ] 自動実行のON/OFF

**あると良い**
- [ ] プリセット（初心者向け/上級者向け等）
- [ ] 設定のテスト機能

---

### 14. 結果表示画面（AI出力の表示）

**必須要素**
- [ ] 結果の明確な表示
- [ ] 結果のコピー機能
- [ ] 結果のダウンロード/エクスポート（該当する場合）
- [ ] 「やり直す」「再生成」ボタン
- [ ] 元の入力への参照（何に対する結果か）

**あると良い**
- [ ] 結果の編集機能
- [ ] 結果の保存機能
- [ ] 結果の共有機能
- [ ] フィードバック（良い/悪い）ボタン
- [ ] 結果の説明（なぜこの結果になったか）

---

## 共通ルール（全画面適用）

### ローディング

- [ ] 0.5秒以上かかる処理にはローディング表示
- [ ] ローディング中は重複操作を防止

### エラー表示

- [ ] エラーは赤色で目立たせる
- [ ] エラー内容は具体的に（「エラーが発生しました」だけはNG）
- [ ] 可能な限り解決方法を提示

### 成功フィードバック

- [ ] 保存/削除/送信の成功は必ずフィードバック
- [ ] トースト通知は3〜5秒で自動消去

### ナビゲーション

- [ ] 現在地がわかる（パンくず、アクティブ状態）
- [ ] 主要画面への導線が常にある

### レスポンシブ

- [ ] モバイルでも操作可能（タップターゲット44px以上）
- [ ] 重要な機能は折りたたまれない

---

## 使い方

ガーディアンAIはこのルール集を読み込み、実装された画面と照合する。

```
チェック方法：
1. 画面の種類を特定（一覧？フォーム？チャット？）
2. 該当パターンの「必須要素」を確認
3. 欠けている要素を報告 or 自動修正
```
